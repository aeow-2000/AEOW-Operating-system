about project

The idea of the project is to build an operating system from scratch, step by step.
We are students from Al-Aqsa University. We will publish on this site all the stages of building the operating system, and we will learn and you all these stages and show you the results and we hope you benefit from us.
The benefits of this project include:
-Win experience.
-Learn new programming languages.
At the end of this project we will come up with an integrated operating system.
